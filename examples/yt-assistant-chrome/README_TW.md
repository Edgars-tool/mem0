# Mem0 助理 Chrome 擴充功能

一個強大的 Chrome 擴充功能，透過 mem0 將 AI 聊天與您的個人知識庫相結合。在不移離頁面的情況下，取得關於影片內容的即時、個人化答案，同時運用您自己的知識和記憶。

## 開發

1. 安裝依賴：
   ```bash
   npm install
   ```

2. 開始開發模式：
   ```bash
   npm run watch
   ```

3. 建置生產版本：
   ```bash
   npm run build
   ```

## 功能

- 直接在 YouTube 中使用 AI 驅動的聊天介面
- 由 Mem0 驅動的記憶功能
- 深色模式支援
- 可自訂選項

## 權限

- activeTab：用於存取目前標籤頁
- storage：用於儲存用戶偏好
- scripting：用於注入內容腳本

## 主機權限

- youtube.com
- openai.com
- mem0.ai

## 功能

- **情境式 AI 聊天**：詢問您正在觀看的影片相關問題
- **無縫整合**：聊天介面置於 YouTube 原生 UI 旁邊
- **由 OpenAI 驅動**：使用 GPT 模型提供智慧回應
- **可自訂**：配置模型設定、外觀和行為
- **未來mem0整合**：根據您的知識提供個人化回應（即將推出）

## 安裝

### 從原始碼（開發者模式）

1. 下載或複製此儲存庫
2. 開啟 Chrome 並前往 `chrome://extensions/`
3. 啟用「開發者模式」（切換開關在右上角）
4. 點擊「載入未封裝」並選擇擴充功能目錄
5. 擴充功能現在應該已安裝並在工具列中可見

### 設定

1. 點擊工具列中的擴充功能圖示
2. 輸入您的 OpenAI API 金鑰（使用擴充功能所需）
3. 如有需要可配置其他設定
4. 前往 YouTube 開始使用助理

## 使用方式

1. 瀏覽任何 YouTube 影片
2. 點擊頁面角落的 AI 助理圖示以開啟聊天介面
3. 詢問關於影片內容的問題
4. AI 會以情境資訊回覆

### 範例提示

- 「你能總結這部影片的主要要點嗎？」
- 「說話者在 5:23 時解釋了什麼？」
- "解釋他們剛才提到的概念"
- 「這與[我正在學習的主題]有何關聯？」
- 「這裡討論的內容有哪些實際應用？」

- **API 設定**：變更模型、調整 tokens、修改溫度
- **介面設定**：控制聊天出現的位置和方式
- **行為設定**：配置自動上下文擷取

## 隱私與資料

- 您的 API 金鑰儲存在瀏覽器本地端
- 影片上下文和字幕僅在您提問時才會在本地處理並傳送至 OpenAI
