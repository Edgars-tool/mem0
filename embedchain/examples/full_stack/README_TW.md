## 🐳 Docker 設定

- 若要使用 docker 設定全端應用程式，請在終端機中此資料夾內執行以下命令。

```bash
docker-compose up --build
```

📝 注意：建置命令可能需要一段時間來安裝所有套件，取決於您的系統資源。

## 🚀 使用說明

- 在瀏覽器中前往 [http://localhost:3000/](http://localhost:3000/) 查看儀表板。
- 在設定中新增您的 `OpenAI API key` 🔑。
- 建立新的機器人，您將被導向至其頁面。
- 在這裡您可以新增資料來源，然後與機器人聊天。

🎉 祝您聊天愉快！ 🎉
